#Optional Scoring Rubric — Plain English 

You do not need a technical background to score these tests.  Use your honest judgment as the viewer.

---

## How to Score

For each test, run the same prompt in both GitHub Copilot and Claude Code. Then answer the questions below for each tool's output. Tally the scores at the end.

---

## Test 1 — Build a Website

**Does it open in a browser without errors?**
- Yes, it loaded fine → 20 points
- It loaded but had minor visual issues → 10 points
- It wouldn't open or was completely broken → 0 points

**Does it look like a real, professional website?**
1 = Looks rough, like a draft &nbsp;&nbsp; 5 = Could be a real business website
Score: _____ × 6 (max 30 points)

**Does it include all the things that were asked for?** (homepage, menu with prices, contact info)
- All three present → 30 points
- Two out of three → 20 points
- One or none → 0 points

**Would you feel comfortable showing this to a customer?**
- Yes → 20 points
- No → 0 points

**Test 1 Max: 100 points**
